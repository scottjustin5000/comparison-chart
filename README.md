# comparison-chart

> react comparison chart

## Install

```bash
npm install --save comparison-chart
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'comparison-chart'
import 'comparison-chart/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
